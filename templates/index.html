Como funciona?

O AWS App Runner é um serviço de aplicações de contêineres totalmente gerenciado que permite criar, implantar e executar aplicações Web e serviços de API em contêineres sem exigir infraestrutura prévia ou experiência em contêineres.

GitHub AWS AppRunner

Por que usar o App Runner?

O AWS App Runner cria e implanta automaticamente aplicações da Web em contêiner, equilibra a carga do tráfego com criptografia, escala para atender às suas necessidades de tráfego e permite configurar como os serviços acessados e comunicar com outras aplicações da AWS em uma Amazon VPC privada.
